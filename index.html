專題緣起

世界上有溝通障礙的人日漸增多，而在台灣也是面臨同樣的情況，所以我們是否可以可以做個什麼東西來幫助他們呢？ 現在，人人幾乎都有一臺手機，手機逐漸成了必需品不離身，可以利用它去做任何事情，若能設計出一個APP為媒介也就是Me語系統，讓原本無法溝通的人們，能使用個人僅有的表達方式彼此可互通心意，這何嘗不是一件既美好又令人感動的事呢？因此，我們便開始展開關於溝通障礙者如何與他人溝通的研究。 人們溝通的方式不只一種，大部分的軟體都只提供單一功能，所以我們希望透過研究，分析出人們常用的溝通方式，將它們整合起來，讓他們使用Me語，建築起一道橋樑打破溝通障礙者和普通人之間溝壑。

專題貢獻

學術界

我們採用三層式系統架構：
(1)前端手機APP(Android + App Inventor)。
(2)中端Agent(XAMPP Web Server)。
(3)後端Bot(Batch批次指令檔 + Python + AI)。
後端為IC套件架構可抽換各個AI功能元件。

事務界

即使在最困難溝通的情況下，例如：聽障者使用手語經由本系統轉成語音與視障的人士溝通。反之，視障者可使用語音經由本系統轉成文字與對方溝通。透過上述Me語的功能，可以讓語言障礙人士及外籍人士願意與一般人進行溝通，分享他們的喜怒哀樂或各種他們想要與他人分享的事情，幫助他們跨出與他人溝通的第一步。而我們之間的談話，透過Me語可以互相轉換成自己方便，對方可以理解的方式來傳送，達到有效的溝通。除此之外，透過這樣的溝通，讓他們知道世界上還是有人在關懷著他們，給予他們一股溫暖的力量。
